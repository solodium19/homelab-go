# homelab-go



## Начинаем

Этот репозиторий создан как домашняя лаборатория для развития своих DevOps навыков, все развёртывания происходили в ветках:

Develop,Stage - в них находится код приложения, я не прогер и просто взял готовый проект от докера с кошечками и собачками, состоящий из схемы Голосовалка - Редис - Воркер - БД - Результаты, для этого приложения поднят кластер K8s на 3 ноды, настроен CI/CD, мониторинг, логирование, балансировщик нагрузки, сертификация, автоскеллинг. Подробнее со всем можно ознакомиться в ветках:

Infrastructure - Инфраструктура нашего кластера
K8s-manifests - Манифесты кубернетеса
CI-pipelines - пайплайны

В каждой ветке кроме Develop,Stage, есть понятные ReadMe с описанием моих действий, столкнувшихся проблем и способов их решения, а так же во многих объяснений тех или иных вещей. Проект ознакомительный, находится в стадии доработки(нету ресурсов на развертывание prod окружения).
P.S все секреты которые хранятся в открытом виде здесь являются недействительными и являются лишь заглушкой
